Algebra Basics for G5

What Is Algebra?

1. Variables and Expressions:

Imagine you have a magic box called a variable. You can put any number inside it. Let's call our variable x.

When we write x + 12, it means "take the number in the box (x) and add 12 to it."

So if x is 5, then x + 12 equals 17.

2. Practice Time!

Try these:

  2. Solve 3z - 8 = 10 to find z.
